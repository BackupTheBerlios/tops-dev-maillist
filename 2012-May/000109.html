Dear TOPS users,
I am glad to announce that thanks to Angelos Tzotzos (Remote Sensing 
Laboratory, National Technical University of Athens) there is now an 
installable package for OpenSUSE. Installing is as easy as:

$ sudo zypper ar 
http://download.opensuse.org/repositories/Application:/Geo/openSUSE_12.1/ GEO
$ sudo zypper refresh
$ sudo zypper install TotalOpenStation

Any report about this package will be very welcome. We will be adding 
this information to the website.

Ciao,
Stefano
