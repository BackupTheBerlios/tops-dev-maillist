Dear TOPS users,
I am happy to announce that we are actively working on adding support 
for the popular GSI format, used by many Leica total stations and 
others. If you own or use a total station that works with GSI, we would 
be grateful if you could help us testing this new feature and making it 
more complete.

Known limitations

Support for raw measurements is still incomplete, namely it lacks 
support for base station points, except in some specific cases when the 
base point coordinates are directly associated with each measurement.

Measurement units (both for distances and angles) are not supported yet.
Acknowledgements

Support for this format was added thanks to Anna Hodgkinson and Hannah 
Petten at the University of Liverpool. Some details of the 
implementation are based on the SurveyTools QGIS plugin developed by 
Stefan Ziegler.

You can see the ongoing work on the documentation at readthedocs.org:

http://totalopenstation.readthedocs.org/en/latest/input_formats/if_leica_gsi.html
