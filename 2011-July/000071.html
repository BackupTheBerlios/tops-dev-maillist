Hi all and especially Maxim,
this is just to say that today I was able to use Total Open Station on
Windows XP successfully to download data from our Zeiss total station.

The "workaround" was to use pyserial 2.4 instead of the most recent
version (2.5) and to install also the pywin32 package for Python
beforehand. Apparently the bug is with pyserial 2.5 (but there's a new
version 2.6 to be released soon, fingers crossed).

If you have any chance to test and confirm that this combination will
work, it will be greatly appreciated and we will change our
documentation accordingly.
