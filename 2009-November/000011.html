We (me and Luca Bianconi) are actively working on it, right now we're in
the process of rewriting the format parser engine to support more
formats from total station devices. The repository is open and you can
get the entire source tree with Mercurial. As the website says, TOPS is
written in the Python programming language, which isn't perhaps very
akin to AutoLISP, but should be quite easy to read, at least.

I would be happy if you could provide some insights on what such a
software should do, where you see room for improvement and new features
you value as important.
