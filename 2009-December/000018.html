Hi,
I did a first attempt at packaging tops today, it is here:
http://pypi.python.org/pypi/totalopenstation

The result is so far incomplete (documentation and scripts are missing,
for one), but you can "pip install totalopenstation" (I strongly suggest
using virtualenv for that purpose) and it works, i.e. you can "import
totalopenstation" inside a Python interpreter.

The current version is marked as 0.2dev, meaning "not released yet". I
think we can tag a 0.2 release when the package is done and ready.

Please test and report,
